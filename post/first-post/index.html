ブログを作ってみた

2024-02-17
 /生活  #etc  #tech

ブログサイトを作った。
IT技術系のやってみた記事や日々の生活などについて投稿する予定。各種フィクションの感想なんかも投稿するかも。
月2回の更新を目標にする。

動機

下記の三点。

  • 文章を書く習慣を身につけたい
  • インターネットで見るITエンジニアには自分のサイトを持っている人が多く自分も持ちたいと常々考えていた
  • SNSで見かけたスライド(Blogを作り、育み、慈しむ ~ Blog Hacks 2024)に触発された

以前も作成しようとしたが、サイトを作成して公開した後一つの記事も書かずにやめてしまった。
今回はとりあえず一つ書いてから公開したので自己ベスト更新と言っていいだろう。言わせてください。

このサイトの構成について

Zola(Rust製のSSG)で生成した静的ファイルをGithub pagesでホストしている。
Zolaの使い方なんかは下記のページを参考にした。

  • 🚛 BlogをZola + Github Pagesに移行した

テーマはemily_zola_themeを利用。 デザインが好みだったため。
ドメインはRoute53で取得、管理している。

コメント機能や検索機能についてはZolaのtemplatesを理解した後に実装してみたい。
そもそも（ITエンジニアとしてはあるまじきことに）HTMLとCSSがよくわかってないので、そちらの勉強が先だけども……。
